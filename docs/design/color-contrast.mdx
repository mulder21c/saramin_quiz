충분한 색상 대비는 사람들이 페이지의 콘텐츠를 명확히 볼 수 있게 합니다.

## 색상 대비

웹 콘텐츠 접근성 지침은 색상 대비 지침을 충족하도록 <a href="http://www.w3.org/TR/UNDERSTANDING-WCAG20/visual-audio-contrast-contrast.html"> 4.5:1의 색상 대비를 권장</a>합니다. 색상 대비율을 검증하기 위해 온라인에서 다양한 색상 대비 검사기를 사용 할 수 있습니다. <a href="http://www.paciellogroup.com/resources/contrastanalyser/">색상 대비 분석기</a>와 <a href="http://snook.ca/technical/colour_contrast/colour.html#fg=33FF33,bg=333333"> 색상 대비 검사 도구</a>는 온라인에서 사용할 수 있는 도구의 예입니다.

다음 두 집 아이콘의 경우, 색상 대비 분석기를 사용하여 색상 대비가 계산되었습니다. 다음 연습을 위해, 여기 지정된 배경 (#405865)과 전경 (#FFFFFF) 값을 입력하도록 <a href="http://snook.ca/technical/colour_contrast/colour.html#fg=33FF33,bg=333333"> 색상 대비 검사 도구</a>를 사용하세요. 도구가 색상 대비율을 보여줄 것입니다. 다음 'x'를 세 번째 아이콘의 대비로 바꾸세요, 예., `<span id='contrastValue'>2000</span>`.
