table은 스크린리더가 표 형식으로 나타나는 정보를 처리하는데 도움이 됩니다.
정보가 table 마크업을 사용하여 나타날 경우, 스크린리더 사용자는 열과 행에 걸쳐 읽을 수 있으며, 열과 행 제목을 들을 수도 있습니다.

## 의미론적 표

다음 표는 의미론적 `table` 마크업을 사용합니다. HTML은 완전히 접근 가능한 표를 만들기 위한 많은 엘리먼트와 어트리뷰트를 제공합니다.
참고: ARIA 1.0은 HTML table과 동등한 것을 포함하지 않습니다; ARIA 1.1에 들어옵니다.
따라서 표 콘텐츠의 2차원 방향키 탐색을 포함하는 대화형 그리드를 구축하는 경우가 아닌 이상 네이티브 HTML을 고수하는 것이 좋습니다.
보이스오버로 다음 표를 테스트 하기 위해:

1. "에디터 출력" 영역에 도달할 때까지 <kbd>Ctrl</kbd> + <kbd>Opt</kbd> + <kbd>Cmd</kbd> + <kbd>h</kbd>를 반복적으로 누르세요.
2. 표 셀을 탐색하기 위해 이제 <kbd>Ctrl</kbd> + <kbd>Opt</kbd> + <kbd>Right Arrow</kbd>를 반복적으로 누르세요.

현 시점에서, 보이스오버는 크롬에서 열 제목을 읽지 않습니다. 하지만 사파리에서는 읽습니다. Windows 스크린리더는 일반적으로 열과 행 제목을 읽습니다.

import { Editor } from "../_util/Editor.mdx";

<Editor>
  {`<table>
  <thead>
    <tr>
      <td></td>
      <th>Week1</th>
      <th>Week2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>Clicks</th>
      <td>100</td>
      <td>90</td>
    </tr>
    <tr>
      <th>Likes</th>
      <td>60</td>
      <td>55</td>
    </tr>
  </tbody>
</table>`}
</Editor>

## 의미론적이지 않은 표 연습

정보가 표 형태일 경우, CSS를 사용하여 표 모양을 만드는 것은
스크린리더 사용자에게 정보를 이해하고 사용하는 것을 거의 불가능하게 만듭니다.
다음 표 데이터를 접근 가능한 HTML table로 변환하세요.

<Editor
  checkExercise={(exercise) => {
    const elementTable = exercise.querySelector("table");
    if (!elementTable) {
      return "Use a semantic table element.";
    }
    const elementTR = exercise.querySelectorAll("tr");
    if (elementTR.length !== 2) {
      return "Are you using semantic tr tags?";
    }
    const elementTH = exercise.querySelectorAll("th");
    if (elementTH.length !== 2) {
      return "Are you using semantic th tags?";
    }
    const elementTD = exercise.querySelectorAll("td");
    if (elementTD.length !== 2) {
      return "Are you using semantic td tags?";
    }
  }}
>
  {`<div className="table">
  <div className="tableHeading">
    <div className="tableHead">Month</div>
    <div className="tableHead">Mood</div>
  </div>
  <div className="tableBody">
    <div className="tableRow">
      <div className="tableCell">January</div>
      <div className="tableCell">happy</div>
    </div>
  </div>
</div>`}
</Editor>
