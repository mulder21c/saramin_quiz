## 사람인 가이드 내용중 포인터 입력 컨트롤의 최소 크기 보장 사이즈를 고르세요.


- 20x20
- 텍스트가 굵지 않고 크기가 24px 이상이거나 텍스트가 굵고 크기가 18px 이상인 경우,  배경색과 글자색의 명도대비가 3:1 이상 되도록 조정해 주세요.
- 행의 간격을 텍스트 사이즈 대비 최소 1.5배 이상으로 설정해 주세요.
- 문단의 간격을 행 간격 대비 최소 1.5배(텍스트 사이즈의 2.25배) 이상으로 설정해 주세요.
